递推（斐波那契）

1.一只小蜜蜂
题目描述：
有一只经过训练的蜜蜂只能爬向右侧相邻的蜂房，不能反向爬行。请编程计算蜜蜂从蜂房a爬到蜂房b的可能路线数。
其中，蜂房的结构如下所示。

Input
输入数据的第一行是一个整数N,表示测试实例的个数，然后是N 行数据，每行包含两个整数a和b(0<a<b<50)。
Output
对于每个测试实例，请输出蜜蜂从蜂房a爬到蜂房b的可能路线数，每个实例的输出占一行。
Sample Input
2
1 2
3 6
Sample Output
1
3
思路分析：
这道题目通过找规律很简单就可以得到这就是一个斐波那契数列，但是使用函数递归会超时，所以改用for循环来进行递推。
注：在该题目中首尾紧贴，则为1，首尾间隔1，则为2，斐波那契返回值中要略去一个1，为1,2,3,5,8……。此处以二者中间间隔数为实参，即fib(0) = 1,fib(1)=2。
由于斐波那契数列后面的数字会急速增长，所以返回的结果要是long long 型，否则会WrongAnswer。

2.爬楼梯
题目描述：
假设你现在正在爬楼梯，楼梯有 n 级。每次你只能爬 1 级或者 2 级，那么你有多少种方法爬到楼梯的顶部？
输入格式
第一行输入一个整数 n(1≤n≤50)，代表楼梯的级数。
输出格式
输出爬到楼梯顶部的方法总数。
Sample Input
5
Sample Output
8
思路分析：
这道题目与小蜜蜂相似，很简单找规律就可以发现每一次包括最后一次上台阶时，都只存在两种方式，通过递归实现就可以得到这同样是一个斐波那契数列。（这道题使用函数递归同样会超时，但可以不使用long long型返回值）。
