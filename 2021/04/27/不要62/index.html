不要62

题目描述：
杭州人称那些傻乎乎粘嗒嗒的人为62（音：laoer）。
杭州交通管理局经常会扩充一些的士车牌照，新近出来一个好消息，以后上牌照，不再含有不吉利的数字了，这样一来，就可以消除个别的士司机和乘客的心理障碍，更安全地服务大众。
不吉利的数字为所有含有4或62的号码。例如：
62315 73418 88914
都属于不吉利号码。但是，61152虽然含有6和2，但不是62连号，所以不属于不吉利数字之列。
你的任务是，对于每次给出的一个牌照区间号，推断出交管局今次又要实际上给多少辆新的士车上牌照了。
Input
输入的都是整数对n、m（0<n≤m<1000000），如果遇到都是0的整数对，则输入结束。
Output
对于每个整数对，输出一个不含有不吉利数字的统计个数，该数值占一行位置。
Sample Input
1 100
0 0
Sample Output
80

题目分析：该题目就是在一个整数串中是否含有4或者连续的62。
最开始使用循环遍历暴力求解，结果超时：

分析过后觉得认为是while循环之中太耗时，因此需要寻找另外的办法来判断每个数是否符合条件。不难得出：若在低位上已经出现了4或者62，则该数字不用再进行逐位判断。例如当 i = 62 不符合条件，flag[62] = 1,则当 i = 622时，因为622 / 10 即62已经不符合题意，所以622也不符合题意，flag[622] = 1。通过这个方法节省出了一些时间，使得该暴力解法得以AC。
