三值排序

题目：
排序是一种很频繁的计算任务。一个实际的例子是，当我们给某项竞赛的优胜者按金银铜牌排序的时候。在这个任务中可能的值只有三种1，2和3。我们用交换的方法把他排成升序的。
写一个程序计算出，计算出的一个包括1、2、3三种值的数字序列，排成升序所需的最少交换次数。
输入第1行为类别的数量N（1≤N≤1000）
输入第2行到第N+1行，每行包括一个数字（1或2或3）。
输出包含一行，为排成升序所需的最少交换次数。

Sample Input
9
2
2
1
3
3
3
2
3
1

Sample Output
4

解题思路：
由题意得，将一个由1,2,3散乱排布组成的数列经过排序之后，分为三个区域，每个区域先后分别为1,2,3。
若根据普通排序的方法，从前向后排序，则免不了把后面的2与前面3交换，之后又再次把交换后的2与更后的1交换，由此这个2则被重复交换了。
为了避免这种重复交换，要确保每个数被交换之后都要到它自己的位置，因此要确定出每个数值的区域，在此处记为s1,s2,s3。
交换这一动作是两个对象之间发生的，而此处有三个对象，任选两个对象，他们之中由于有第三个对象元素的存在，则它们之中非自己元素的个数并不一定相等，因此不能直接交换。
因此要从这三个区域中挑出一个区域与另两个区域分离开进行操作。
先挑一个区域1，计算出其中非己的元素w1，将它们分给另两个区域之中，此时那的两个区域中非己元素是相同的，记为ans2，则最终答案即为 w1 + ans2。
所以此时的题目变为了求ans2:
*****从1区域分别向2,3区域中交换的数字个数未知，因此不能直接求出w2。
由于1区域已经排好，所以只用求2,3区域中非己元素，此时2,3区域中的非己元素只能是对方区域的元素，因此找出2区域中3的个数w2,和3区域中2的个数w3即可。
那么ans2应该取w2和w3中的那一个呢：假设s2中的3较多，即 w2>w3，则s1中的2比较多，若以把s2复原为目标（即令ans2 = w3），则将s1中的s2重复交换。
因此只需要在w2和w3之中选出较大值作为ans即可，最终与s1相加即为最终结果。
